🎮 Juego de Adivinanza en Python
📌 Descripción

Este proyecto es un juego en consola desarrollado en Python, donde el usuario debe adivinar un número o una palabra.
Incluye validaciones y repetición de intentos para mejorar la experiencia.

🚀 Tecnologías utilizadas
Python 🐍
⚙️ Funcionalidades
Entrada de datos por el usuario
Validación de respuestas
Uso de bucles para múltiples intentos
Mensajes interactivos
📂 Estructura del proyecto
01-usuario.py → Manejo básico de usuario
02-calculadora.py → Calculadora simple
03-juego.py → Juego de adivinanza
04-objetos.py → Programación orientada a objetos
05-contraseña.py → Gestión de contraseña
